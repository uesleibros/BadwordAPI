👋 Olá, Conheça a nossa API!

Trata-se de uma api de palavrões/xingamentos. É bem simples, porém é bem completa.

Feita principalmente para você moderar o que pessoas estão falando em seu jogo, servidor do discord, aplicativo de chat ou qualquer coisa do gênero.

Atualmente ela se encontra em seu estado de desenvolvimento, é uma API recente e tem muita a adicionar ainda.

Acompanhe o quanto já fizemos de nossa API:

99%



🧠 Rota Principal -> /badwords

Toda a API ocorre nessa Rota. Apenas ao entrar na rota você obtem a lista de xingamentos que a API possui no momento.

Parâmetros (1)

O parâmetro ?lang serve para escolher a língua em que os xingamentos iram ser imprimidos, no momento temos suporte a apenas 2 línguas: Português e Inglês.

Como usar

Para aplicar o parâmetro na API, basta adicionar ?lang=língua. Para Português usa: pt-br e para Inglês usa: en

Exemplo

Para Inglês

https://badwordapi.uesleidev.repl.co/badwords?lang=en

Para Português

https://badwordapi.uesleidev.repl.co/badwords?lang=pt-br ou https://badwordapi.uesleidev.repl.co/badwords
